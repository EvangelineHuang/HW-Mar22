I have generated five graphs of the grades of the Crafty Penguin. In order to generate each graphic, I used five different CSV files to make it easier to get data. In order to find out how the penguin behaves in quizz and homework, I used line chart to see how its grades change over time. Then in order to learn about what is the most frequent grade it has, I used two bar charts to calculate its frequency of each grade of homework and quizzes.

When creating each graphic, I first import the data from CSV files and then bind the data to each rect or to the path which is used to create the line chart. Be default, the line chart has black shadows, so I set the fill to none and give the line chart a stroke. After creating the shape of the charts, I created the Axises of each chart.
